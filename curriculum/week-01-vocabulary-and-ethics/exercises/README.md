# Week 1 — Exercises

Short, focused drills. Each one should take 20–40 minutes. Do them in order; later ones assume earlier ones.

## Index

1. **[Exercise 1 — Glossary in your own words](exercise-01-glossary.md)** — define every Week-1 term without copying. (~30 min)
2. **[Exercise 2 — FASTA by hand](exercise-02-fasta-by-hand.py)** — parse a FASTA file in pure Python, no Biopython. (~35 min)
3. **[Exercise 3 — Public-data inventory](exercise-03-data-inventory.md)** — catalog the datasets you will use across C10. (~30 min)

## How to work the exercises

- Read the prompt. Skim, do not memorize.
- **Type the code yourself.** Do not copy-paste. Muscle memory is the entire point.
- Run it. See the output. Read the error if it crashed.
- If you get stuck for more than 10 minutes, peek at the inline hints.
- Only check the official solution after you have made it run.

There are no solutions checked in. Solutions live in forks and Gists from other learners.
